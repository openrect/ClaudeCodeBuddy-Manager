# ClaudeCodeBuddy Manager

[English README](./README.md)

一个基于 Tkinter 的 Claude Code宠物指定器：可视化挑选宠物物种、稀有度与外观组合，自动生成对应 `userId`，并一键写入 `.claude.json`。

## 使用方式（Windows）
### 方式 1：直接运行可执行文件

1. 下载运行 `dist/ClaudeCodeBuddy-Manager-win-x64.exe`
2. 完成筛选条件
3. 点击应用按钮写入配置
